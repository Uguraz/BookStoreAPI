# Bookstore API

Et eksamensprojekt til softwarekvalitet. Dette er en simpel boghandel med teststrategier som TDD, BDD, integrationstest og end-to-end test.

## Funktioner
- Tilføj og hent bøger
- Læg ordrer
- Brug JWT-login
- Automatisk test og CI med GitHub Actions

## Sådan køres det
```bash
npm install
npm start
